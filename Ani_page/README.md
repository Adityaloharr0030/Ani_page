# Ani Editor  
  
A robust, web-based development environment designed for efficient coding and project management.  
  
## Key Features  
* **Smart Code Completion**: Automated suggestions.  
* **File Management**: Full control over files.  
* **Integrated Terminal**: Execute commands directly.  
  
## Installation and Setup  
1. Clone the repository.  
2. Run 'npm install'.  
3. Run 'start.bat'.

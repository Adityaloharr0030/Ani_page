# Ani Editor

A full-featured web-based code editor with intelligent code assistance, file management, and integrated terminal capabilities.

## Overview

Ani Editor is a modern IDE built as a web application, providing developers with a powerful coding environment accessible through any browser.

## Features

### Code Editing
- Smart Code Completion with context-aware suggestions
- Syntax Highlighting for multiple programming languages
- Multi-file Editing capabilities
- Automatic Code Formatting

### File Management
- Visual Tree View navigation
- Full CRUD operations for files and directories
- File Upload and Download support

### Development Tools
- Integrated Terminal
- Live Preview for web applications
- Git integration

## Technology Stack

**Frontend:** React.js, Vite, Monaco Editor  
**Backend:** Node.js, Express.js, WebSocket

## Getting Started

### Prerequisites
- Node.js (v16.0.0+)
- npm (v8.0.0+)

### Installation

```bash
git clone https://github.com/Adityaloharr0030/Ani_page.git
cd Ani_page
npm install
```

### Running

**Windows:** Run `start.bat`  
**Manual:** `npm start`

Access at:
- Frontend: http://localhost:5173
- Backend: http://localhost:3000

## Project Structure

```
Ani_page/
├── ai-code-editor/
│   ├── client/       # React frontend
│   └── server/       # Node.js backend
├── start.bat

```

## Usage

1. Create files via right-click in file explorer
2. Edit code in the main editor
3. Use integrated terminal for commands
4. Preview web apps with live preview
